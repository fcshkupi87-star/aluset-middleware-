# ALUSET Middleware – Chatbase + Microsoft Translator

Ky projekt është një server i vogël Node.js (Express) që pranon mesazhe,
i përkthen me Microsoft Translator API, ia dërgon Chatbase dhe kthen
përgjigjen e përkthyer përsëri te përdoruesi.

---

## Struktura e fajllave

aluset-middleware/
 ├── server.js
 ├── package.json
 └── .env.example

---
